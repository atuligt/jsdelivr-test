UtilHelpers.js Demo

This page demonstrates the functionality of the UtilHelpers library loaded via jsDelivr.

Format Number

Adds commas as thousand separators to numbers.

UtilHelpers.formatNumber(1234567) // Returns "1,234,567"

Random Integer

Generates a random integer between min and max values (inclusive).

UtilHelpers.randomInt(1, 100) // Returns a random number between 1 and 100

Debounce

Type quickly in the field below and see how the debounced function only executes after you stop typing.

Waiting for you to stop typing...
const debouncedFunction = UtilHelpers.debounce(myFunction, 500);

Slugify

Converts a string into a URL-friendly slug.

UtilHelpers.slugify("Hello World! This is a test.") // Returns "hello-world-this-is-a-test"

Shuffle Array

Randomly shuffles the elements of an array.

UtilHelpers.shuffleArray([1, 2, 3, 4, 5]) // Returns array with elements in random order

Current Date

Gets the current date in YYYY-MM-DD format.

UtilHelpers.getCurrentDate() // Returns current date in YYYY-MM-DD format
